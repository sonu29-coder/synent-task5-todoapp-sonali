# ✨To-Do List ✨

## 📌 Objective
A stylish task management application built with **HTML**, **CSS**, and **JavaScript**.  
This project demonstrates DOM manipulation, localStorage usage, and elegant UI design.

---

## 🎨 Features
- Add new tasks  
- Delete tasks  
- Mark tasks as completed  
- Persistent storage using localStorage (tasks remain after refresh)  
- Royal-themed UI with deep blue and golden accents  

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the app  
- **CSS3** – Styling with gradients, shadows, and royal colors  
- **JavaScript (ES6)** – Logic for task management and localStorage  

---

todo-app/

## 🚀 How to Run
1. Clone or download this repository.  
2. Open the project folder in **VS Code**.  
3. Run `index.html` in your browser.  
4. Start adding tasks — they’ll stay even after refresh!
